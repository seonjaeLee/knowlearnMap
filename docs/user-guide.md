# 사용자 가이드

KNOWLEARN MAP 앱 내부 메뉴에서 열리는 사용자 안내 문서입니다.

## 메모 메뉴

상단 메모 메뉴에서 아래 항목을 사용할 수 있습니다.

- `새 메모 추가`: 새로운 메모를 생성합니다.
- `메모 목록`: 저장된 메모 목록을 확인하고 원하는 메모를 엽니다.
- `모두 숨기기`: 현재 화면에 열려 있는 메모를 일괄 숨깁니다.
- `로그인 시 자동 실행`: 로그인할 때 메모 기능을 자동으로 실행할지 설정합니다.
- `사용 가이드`: 이 문서를 엽니다.
- `종료`: 메모 메뉴를 닫거나 기능을 종료합니다.

## 메모 목록 팝업 사용 방법

1. 메모 메뉴에서 `메모 목록`을 선택합니다.
2. 목록에서 메모를 선택하면 해당 메모가 열립니다.
3. 필요 없는 메모는 닫거나 숨김 처리해 화면을 정리할 수 있습니다.

## 권장 사용 흐름

1. `새 메모 추가`로 작업 메모를 만듭니다.
2. 작업 중에는 `메모 목록`에서 필요한 메모만 열어 사용합니다.
3. 작업이 끝나면 `모두 숨기기`로 화면을 정리합니다.
4. 반복 사용 시 `로그인 시 자동 실행`을 켜 두면 편리합니다.
